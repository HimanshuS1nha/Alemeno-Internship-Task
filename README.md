# Alemeno Internship Task

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npm run dev
   ```

3. For deploying, build the app using the command below and then host the dist folder

   ```bash
    npm run build
   ```
